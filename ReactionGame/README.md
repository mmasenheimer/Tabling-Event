# Reaction Game
A simple reaction game built using HTML, CSS, and JavaScript.
## How to Play
1. Open `main.html` in your browser.
2. Enter your name and click to start.
3. Wait for the screen to turn green.
4. Click as fast as you can!
5. Your reaction time will be displayed.
## How to Run
Download the project folder and open `main.html` in any web browser.
